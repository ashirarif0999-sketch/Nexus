# Plan: Implement Squiggle SVG Scroll-Driven Animation

## Goal
Add a scroll-driven animation to the squiggle SVG that starts when the "Full Service Fund Management" card's left side enters the viewport. The animation has two phases: first animating up quickly while scrolling down, then animating down normally with a curve.

## Current State Analysis
- The squiggle.svg file exists but is not used in the HTML
- GSAP and ScrollTrigger are already loaded in the HTML
- Full-service section is at line 3065-3085 in landingpage.html
- The card has class "full-service-card" with grid layout (1fr 1fr)
- SVG dimensions: 1291x1455 viewBox, tall vertical squiggle with gradient colors

## Implementation Steps

1. **Add Squiggle SVG to HTML**
   - Insert an <img> tag referencing squiggle.svg inside the full-service section, positioned absolutely on the left
   - Add class "squiggle-svg" for styling and animation targeting
   - Place it after the full-service-card div

2. **CSS Positioning and Styling**
   - Position absolutely: left: -100px, top: 50%, transform: translateY(-50%)
   - Set width: 200px, height: auto to make it visible but not overwhelming
   - Add opacity: 0.7 for subtle effect
   - Ensure z-index is below content but above background

3. **ScrollTrigger Animation Setup**
   - Create ScrollTrigger with trigger: ".full-service-card", start: "left center"
   - Use scrub: 1 for smooth animation
   - Animation timeline with two phases:
     - Phase 1 (0-50% progress): translateY from 0 to -200px (fast upward)
     - Phase 2 (50-100% progress): translateY from -200px to 100px with ease curve

4. **JavaScript Implementation**
   - Add script tag at end of body with GSAP animation code
   - Use gsap.timeline() with ScrollTrigger
   - Add markers: true temporarily for debugging

### 3. JavaScript Changes
Add before closing `</body>` tag:

```javascript
document.addEventListener('DOMContentLoaded', function() {
    // Squiggle Scroll Animation
    gsap.registerPlugin(ScrollTrigger);

    gsap.to('.squiggle-svg', {
        scrollTrigger: {
            trigger: '.full-service-card',
            start: 'left center',
            end: 'right center',
            scrub: 1,
            markers: false,
        },
        y: -300, // Fast upward movement
        ease: 'none'
    });

    // Second phase animation
    gsap.to('.squiggle-svg', {
        scrollTrigger: {
            trigger: '.full-service-card',
            start: 'center center',
            end: 'bottom center',
            scrub: 1,
        },
        y: 150, // Downward with curve
        rotation: 5,
        ease: 'power2.out'
    });
});
```
